EnCORE Online Printing Client
(Terminal Works)

http://www.terminalworks.com/remote-desktop-printing/downloads

Directions

  • Exit EnCORE using File then Exit or click the red X in the upper right corner of EnCORE.
  • Download the remote desktop printing client (Red Client Version) from the Terminal Works link.
  • Install the Terminal Works client.
  • Use your local printer to print a test page.
  • Log back into EnCORE and print to the TSDefault printer.
  • If the TSDefault printer does not have the green check then you can right click and set as your default.
